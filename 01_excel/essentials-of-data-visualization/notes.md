## 1. Data Visualization Basics
- Data visualization is the graphical representation of data to identify patterns, trends, and insights.
- Effective visuals reduce cognitive load and improve decision-making.
- Poor visualization can mislead users even if data is correct.

## 2. Types of Data
- Quantitative Data: Numerical values used for measurement and analysis.
- Categorical Data: Group-based or label-based data.
- Samples are subsets of a population used for analysis.

## 3. Charts and Their Use Cases
- Bar / Column Chart: Comparison across categories.
- Line Chart: Trends over time.
- Scatter Plot: Relationship or correlation between two variables.
- Pie / Donut Chart: Proportion of a whole (use sparingly).
- Histogram: Frequency distribution of numerical data.

## 4. Tables and Data Organization
- Tables help organize raw data into readable formats.
- Useful for identifying direct relationships between variables.
- Sorting and filtering improve analysis efficiency.

## 5. Dashboards
- Dashboards combine multiple visuals on one screen.
- Used to track KPIs and support quick decision-making.
- Should be simple, focused, and insight-driven.

## 6. Excel Techniques Used
- Pivot Tables for aggregation
- Conditional Formatting for pattern detection
- Filtering and Sorting for data refinement
- Slicers for interactive dashboards

## 7. Visualization Best Practices
- Choose the right chart for the data
- Avoid excessive colors and decorations
- Label axes and legends clearly
- Maintain high data-ink ratio
