# Essentials of Data Visualization using MS Excel

**Domain:** Data Visualization  
**Tool:** Microsoft Excel  
**Status:** Completed  
**Artifacts Available:** Certificate, Notes, Quiz Results, Mini Project, Charts

---

## Overview
This course focused on building a strong foundation in data visualization principles using Microsoft Excel. The emphasis was on converting raw data into meaningful visuals and dashboards that support analysis and decision-making.

---

## Topics Covered
- Fundamentals of data visualization
- Types of data: quantitative and categorical
- Use of tables for data organization
- Chart selection and use cases
  - Line charts
  - Bar / Column charts
  - Scatter plots
  - Pie / Donut charts
  - Histograms
- Excel dashboards for performance monitoring
- Data modification techniques in Excel
  - Filtering and sorting
  - Conditional formatting
- Quantitative analysis concepts
  - Frequency distributions
  - Relative frequency
- Descriptive statistics
  - Measures of location and variability
  - Correlation and covariance
