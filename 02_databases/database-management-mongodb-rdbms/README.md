# Database Management (MongoDB & RDBMS)

**Domain:** Databases  
**Status:** Completed  
**Artifacts Available:** Certificate

## Overview
This course introduced the fundamentals of database management systems, covering both relational databases (RDBMS) and NoSQL databases (MongoDB). The focus was on understanding database concepts, data modeling, and differences between structured and unstructured data storage.

## Topics Covered
- Fundamentals of Database Management Systems (DBMS)
- Relational Databases (RDBMS)
  - Tables, rows, and columns
  - Primary and foreign keys
  - Basic normalization concepts
- NoSQL Databases
  - MongoDB document-based data model
  - Collections and documents
  - Schema flexibility
- Comparison between RDBMS and MongoDB
  - Use cases
  - Scalability considerations
  - Structured vs semi-structured data

## Learning Outcome
- Gained conceptual understanding of how relational and NoSQL databases store and manage data
- Understood when to use RDBMS versus MongoDB based on application requirements
- Built foundational knowledge for backend development and data-driven applications
